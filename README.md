# lcp-utils

Utilities for Lens Correction Profile (LCP) workflows.

## Development

This project targets Python 3.12.

```powershell
conda create -n lcp-utils python=3.12
conda activate lcp-utils
conda install pytest
python -m pip install -e .
python -m pytest
```
